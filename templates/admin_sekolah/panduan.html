Cara Pengajuan Data Sekolah

1. Pilih Halaman Data Sekolah
2. Klik Tombol Tambah Data
3. Isi Data Sekolah

*Untuk Pengisian Poligon di peta
1. Pilih Fullscreen
2. Cari Lokasi Sekolah
3. Tekan Tanda + Untuk Memperbesar dan Tanda - Untuk Memperkecil 4. Pilih Draw a Polygon
5. Gambar Poligon Sesuai Bentuk Sekolah
6. Klik FInish
7. Tekan ESC Pada Keyboard atau Exit Fullscreen Untuk Keluar Dari Halaman Fullscreen Peta

Cara Melihat Data Sudah Diterima atau Belum

1. Data yang Sudah Diterima Akan Ditampilkan Pada Halaman Dashboard
2. Jika Data Tidak Ada, Maka Data Mungkin Ditolak Oleh Admin Utama
3. Data Paling Lambat Diproses 3 Hari Kerja

Cara Pengajuan Akreditasi Sekolah

1. Pilih Halaman Pengajuan Akreditasi Untuk SMA atau SMK
2. Isi Data Pengajuan yang Telah Disediakan
3. Tekan Kirim
